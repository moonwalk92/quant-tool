## ⚙️ 配置参数

### 默认配置

| 参数 | 值 | 说明 |
|------|-----|------|
| 初始资金 | $10,000 | 虚拟资金 |
| 单笔风险 | 1% | 每笔交易风险 |
| 止盈/止损 | 20 点 | 固定止盈止损 |
| 挂单间隔 | 5 点 | 挂单距离 |
| 最大回撤 | 20% | 超过自动停止 |

### 修改配置

编辑 `src/server.js`：

```javascript
const engine = new TradingEngine({
  initialCapital: 10000,    // 修改初始资金
  riskPerTrade: 0.01,       // 修改单笔风险
  takeProfitPoints: 20,     // 修改止盈
  stopLossPoints: 20,       // 修改止损
  maxDrawdown: 0.20,        // 修改最大回撤
  // ...
});
```

修改后重启服务。

## ⚠️ 重要提示

### 模拟模式

**当前版本使用模拟价格**，不是真实交易！

适合：
- ✅ 学习量化交易
- ✅ 测试策略
- ✅ 演示给别人看
- ✅ 熟悉系统

### 真实交易

如需真实交易，需要：

1. **MT4 交易账户** - 在券商开户
2. **MT4 桥接服务** - 连接 MT4
3. **充分测试** - 至少 1-2 周模拟
4. **风险准备** - 用闲钱交易

详细查看：[README-REAL.md](README-REAL.md)

**A:**
