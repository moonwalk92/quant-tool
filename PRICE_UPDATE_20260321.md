## 🔄 如何更新价格

### 手动校准（推荐临时方案）

编辑 `src/price-api.js` 中的 `getBasePrice()` 函数：

```javascript
getBasePrice(symbol) {
  const basePrices = {
    'XAUUSD': 4690,  // 在这里更新
    // ...
  };
  return basePrices[symbol] || 4690;
}
```

然后：
1. 查看真实市场价格（如 Kitco、GoldPrice.org）
2. 更新基准价格
3. 提交到 GitHub
4. Railway 自动部署

### 频率建议

- **测试阶段**: 每天或每周更新一次
- **生产环境**: 必须使用真实 API
