# Quant Tool EA 安装指南

## 概述

Quant Tool EA 是一个 MetaTrader 4 专家顾问，可以直接在 MT4 中运行量化交易策略。

## 文件结构

```
QuantToolEA/
├── QuantToolEA.mq4          # EA 源代码
├── QuantToolEA.ex4          # 编译后的 EA（需要编译）
└── README.md                # 本文件
```

## 安装步骤

### 第 1 步：编译 EA

#### 方法 A：使用 MT4 MetaEditor（推荐）

1. **打开 MetaEditor**
   - 在 MT4 中按 `F4` 或点击 工具 → MetaQuotes Language Editor

2. **打开源代码**
   - 文件 → 打开
   - 选择 `QuantToolEA.mq4`

3. **编译**
   - 点击编译按钮或按 `F7`
   - 检查是否有错误
   - 成功后会生成 `QuantToolEA.ex4`

#### 方法 B：在线编译

如果无法使用 MetaEditor，可以使用在线编译服务（需要 .mq4 文件）。

### 第 2 步：安装 EA 到 MT4

1. **打开 MT4 数据文件夹**
   - MT4 → 文件 → 打开数据文件夹

2. **复制 EA 文件**
   - 将 `QuantToolEA.ex4` 复制到：
     ```
     MQL4/Experts/QuantToolEA.ex4
     ```

3. **刷新 MT4**
   - 在 MT4 的导航器中右键 → 刷新
   - 或重启 MT4

4. **验证安装**
   - 在导航器 → 专家顾问 中应该能看到 `QuantToolEA`

### 第 3 步：安装 ZeroMQ 库（可选，用于高级功能）

如果需要与 Python 桥接服务通信，需要安装 ZeroMQ 库：

#### 下载 dzmt4mq4.dll

1. 从 GitHub 下载：
   https://github.com/dzmitry-kot/ZeroMQ-MT4

2. 或使用预编译版本：
   ```bash
   # 下载链接（请检查最新版本）
   https://github.com/dzmitry-kot/ZeroMQ-MT4/releases
   ```

3. **复制 DLL 文件**
   - 将 `dzmt4mq4.dll` 复制到：
     ```
     MQL4/Libraries/dzmt4mq4.dll
     ```

4. **启用 DLL 导入**
   - MT4 → 工具 → 选项 → 专家顾问
   - ✅ 允许 DLL 导入

### 第 4 步：配置 EA

1. **将 EA 添加到图表**
   - 在导航器中拖动 `QuantToolEA` 到图表
   - 或右键图表 → 专家顾问 → QuantToolEA

2. **配置参数**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| BridgeHost | 127.0.0.1 | Python 桥接服务地址 |
| BridgePort | 5555 | Python 桥接服务端口 |
| InitialCapital | 10000 | 初始资金（美元） |
| RiskPerTrade | 0.01 | 单笔风险（1%） |
| TakeProfitPoints | 20 | 止盈点数 |
| StopLossPoints | 20 | 止损点数 |
| PendingOrderInterval | 5 | 挂单间隔（点） |
| MaxDrawdown | 0.20 | 最大回撤（20%） |

3. **启用自动化交易**
   - 点击 MT4 工具栏的"算法交易"按钮
   - 确保按钮是绿色的（启用状态）

4. **验证运行**
   - 图表右上角应该显示 EA 图标
   - 图表上应该显示 Quant Tool EA 的信息
   - 点击"启动交易"按钮开始

### 控制按钮

EA 会在图表上添加三个按钮：

- **启动交易** - 开始量化交易策略
- **停止交易** - 停止交易并撤销所有挂单
- **重置** - 重置回撤限制（触发后使用）

## 交易逻辑

### 启动流程

1. 点击"启动交易"
2. EA 发送双向市价单（做多 + 做空）
3. 设置双向挂单（成交价 ±5 点）
4. 每秒监控价格和持仓

### 挂单逻辑

- 做多挂单 = 成交价 + 5 点
- 做空挂单 = 成交价 - 5 点
- 挂单触发后自动重新设置

### 止盈止损

- 止盈：20 点
- 止损：20 点
- 自动计算，无需手动设置

### 风险控制

- 单笔风险：初始资金的 1%
- 最大回撤：20%（触发后自动停止）
- 手数范围：0.01 - 1.0 手

## 故障排除

### 问题 1：EA 不执行交易

**可能原因：**
- 算法交易未启用
- MT4 未登录
- 市场已收盘

**解决方法：**
1. 检查 MT4 工具栏的"算法交易"按钮是否为绿色
2. 检查 MT4 是否已登录交易账户
3. 确认交易品种可交易

### 问题 2：订单发送失败

**可能原因：**
- 余额不足
- 手数超出限制
- 点差过大

**解决方法：**
1. 检查账户余额
2. 调整 RiskPerTrade 参数
3. 避开高波动时段

### 问题 3：ZeroMQ 连接失败

**可能原因：**
- Python 桥接服务未启动
- 端口被占用
- 防火墙阻止

**解决方法：**
1. 启动 Python 桥接：`python mt4_bridge.py`
2. 测试连接：`python test-bridge.py`
3. 检查防火墙设置

### 问题 4：EA 编译错误

**可能原因：**
- MT4 版本过旧
- 缺少库文件
- 代码语法错误

**解决方法：**
1. 更新 MT4 到最新版本
2. 检查是否缺少导入的库
3. 查看 MetaEditor 的错误信息

## 性能优化

### MT4 设置

1. **减少图表数量** - 只保留必要的图表
2. **关闭新闻** - 工具 → 选项 → 新闻 → 禁用
3. **降低日志级别** - 只记录重要信息

### EA 参数调整

根据交易品种调整：

**XAUUSD (黄金):**
- TakeProfitPoints: 20-30
- StopLossPoints: 20-30
- PendingOrderInterval: 5-8

**EURUSD:**
- TakeProfitPoints: 15-25
- StopLossPoints: 15-25
- PendingOrderInterval: 3-5

**GBPJPY:**
- TakeProfitPoints: 25-40
- StopLossPoints: 25-40
- PendingOrderInterval: 8-12

## 安全提示

⚠️ **重要提醒：**

1. **先用模拟账户测试** - 至少测试 1-2 周
2. **从小资金开始** - 初始资金建议 $1000-5000
3. **监控系统运行** - 定期检查 EA 状态
4. **设置合理参数** - 根据风险承受能力调整
