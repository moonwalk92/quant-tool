# 🚀 真实交易前端部署指南

## ✅ 已完成功能

### 前端页面
- **真实交易页面**: `/real-trading.html`
- **完整的交易界面**: 账户信息、价格显示、持仓管理、风险控制
- **MT5 连接功能**: 弹窗输入账户信息，安全连接
- **实时数据推送**: WebSocket 实时更新价格、账户、持仓

### 后端 API
- `POST /api/real/connect` - 连接 MT5 账户
- `GET /api/real/status` - 获取交易状态
- `POST /api/real/start` - 启动真实交易
- `POST /api/real/stop` - 停止真实交易
- `GET /api/real/price` - 获取实时价格

## 📋 部署步骤

### 1. Railway 部署

#### 环境变量配置
在 Railway 项目设置中添加以下环境变量：

```bash
# MT5 账户信息（真实交易必填）
MT5_LOGIN=你的账户号
MT5_PASSWORD=你的交易密码
MT5_SERVER=你的服务器名称

# 交易参数
TRADING_SYMBOL=XAUUSD
RISK_PER_TRADE=0.01
MAX_DRAWDOWN=0.20

# 数据源
USE_REAL_MT4=true
USE_MT4_BRIDGE=false
```

#### 部署检查清单
- [ ] 确认 Railway 项目已连接 GitHub 仓库
- [ ] 检查 `package.json` 中的启动脚本
- [ ] 添加所有必需的环境变量
- [ ] 重启 Railway 部署

### 2. 本地测试

```bash
# 安装依赖
npm install

# 启动服务器
npm start

# 访问页面
# 模拟交易：http://localhost:3000
# 真实交易：http://localhost:3000/real-trading.html
```

## 🎯 使用流程

### 步骤 1：访问真实交易页面
打开浏览器访问：https://quant-tool-production.up.railway.app/real-trading.html

### 步骤 2：连接 MT5 账户
1. 点击「🔌 连接 MT5」按钮
2. 在弹窗中填写：
   - MT5 账户号
   - MT5 密码
   - 服务器名称
3. 阅读风险警告并确认
4. 点击「确认连接」

### 步骤 3：配置交易参数
- **交易品种**: XAUUSD (黄金)、XAGUSD (白银) 等
- **单笔风险**: 0.1% - 5% (推荐 1%)
- **止盈点数**: 5 - 100 点 (推荐 20 点)
- **止损点数**: 5 - 100 点 (推荐 20 点)

### 步骤 4：启动交易
1. 确认配置无误
2. 点击「▶ 启动交易」
3. 系统自动计算手数并下单

### 步骤 5：监控交易
- **实时价格**: Bid/Ask/Spread
- **账户信息**: 余额/净值/可用资金
- **当前持仓**: 方向/手数/盈亏
- **风险控制**: 回撤百分比/进度条

### 步骤 6：停止交易
点击「⏹ 停止交易」按钮，系统自动平仓所有持仓。

## 🔒 安全特性

### 1. 风险控制
- **最大回撤**: 20% (可配置)
- **单笔风险**: 1% (可配置)
- **自动止损**: 达到回撤阈值自动停止

### 2. 连接安全
- **环境变量**: 账户信息通过环境变量管理
- **会话管理**: 连接状态保存在服务器内存
- **风险确认**: 连接前强制显示风险警告

### 3. 数据保护
- **不存储密码**: 密码仅用于连接，不持久化
- **加密传输**: 使用 HTTPS/WSS 加密通信
- **日志脱敏**: 日志中不显示完整账户信息

## ⚠️ 注意事项

### 1. 网络要求
- 需要稳定的网络连接
- 建议使用 VPS 部署 (24/7 运行)
- 避免在网络不稳定时交易

### 2. 资金要求
- 建议至少 $1000 起始资金
- 只用能承受损失的资金
- 充分测试后再实盘

### 3. 测试建议
- 先用模拟账户测试
- 测试至少 2 周
- 确认策略稳定后再实盘

### 4. 监控建议
- 定期检查交易状态
- 设置价格提醒
- 关注账户净值变化

## 🛠️ 故障排查

### 无法连接 MT5
1. 检查账户信息是否正确
2. 确认 MT5 终端已运行
3. 检查网络连接
4. 查看服务器日志

### 价格不更新
1. 检查 WebSocket 连接
2. 确认数据源配置
3. 检查是否是交易时间

### 交易无法启动
1. 确认已连接 MT5
2. 检查账户资金是否充足
3. 查看错误提示信息
