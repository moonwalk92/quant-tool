# 量化交易工具 - MT4 网页版

一个基于 MT4 的量化交易系统，实现双向多开、顺势挂单、风险控制等功能。

## 功能特性

### ✅ 核心功能

- **实时数据跟踪** - 每秒获取 MT4 实时报价，确保价格同步
- **双向多开** - 首次启动发送双向市价单（做多 + 做空），挂单间隔 5 个点
- **止盈止损** - 每单固定 20 点止盈/止损
- **顺势挂单** - 交易完成后，以上一笔成交价为基准，间隔 5 点挂双向止损挂单
- **仓位管理** - 按"单笔风险 1% 初始资金"计算手数，限制 0.01-1.0 手
- **回撤控制** - 实时计算当日回撤，超过 20% 自动停止交易

### 📊 网页界面

- 实时价格显示
- 当前权益/回撤监控
- 持仓和挂单管理
- 交易历史记录
- 一键启动/停止/重置

## 快速开始

```bash

```

### 2. 启动服务

```bash
npm start
```

### 3. 访问界面

打开浏览器访问：http://localhost:3000

## 配置说明

在 `src/server.js` 中可以修改以下参数：

```javascript
{
  symbol: 'XAUUSD',              // 交易品种
  initialCapital: 10000,         // 初始资金（美元）
  riskPerTrade: 0.01,            // 单笔风险（1%）
  minLots: 0.01,                 // 最小手数
  maxLots: 1.0,                  // 最大手数
  takeProfitPoints: 20,          // 止盈点数
  stopLossPoints: 20,            // 止损点数
  pendingOrderInterval: 5,       // 挂单间隔（点）
  maxDrawdown: 0.20,             // 最大回撤（20%）
  pointValue: 0.1,               // 点值（XAUUSD: 1 点=0.1 美元）
  contractSize: 100              // 合约大小
}
```

## MT4 集成

### ✅ 已集成方案

本项目已内置 **Python + ZeroMQ + MetaTrader4** 桥接方案！

### 快速启动

#### macOS / Linux

```bash
./start-mt4-bridge.sh
```

#### Windows

```bash
cd mt4-bridge
start.bat
```

### 启动选项

脚本会提供两个选项：

1. **仅启动 Node.js 服务器** - 使用模拟价格（适合测试）
2. **启动 Python MT4 桥接 + Node.js** - 连接真实 MT4（需要 MT4 终端运行）

### 真实 MT4 连接配置

如果需要连接真实 MT4 账户，编辑 `mt4-bridge/mt4_bridge.py`：

```python
bridge = MT4Bridge(
    host='localhost',
    port=5555,
    mt4_login='你的 MT4 账号',
    mt4_password='你的 MT4 密码',
    mt4_server='你的 MT4 服务器'
)
```

## 交易逻辑流程

```
启动交易
    ↓
发送双向市价单（做多 + 做空）
    ↓
设置双向挂单（成交价 ±5 点）
    ↓
实时监控价格（每秒）
    ↓
┌─────────────────────────────────┐
│ 价格更新                        │
│   ↓                             │
│ 检查挂单是否触发 → 触发则开仓    │
│   ↓                             │
│ 检查持仓止盈止损 → 触发则平仓    │
│   ↓                             │
│ 平仓后重新设置挂单              │
│   ↓                             │
│ 检查回撤 → 超过 20% 则停止        │
└─────────────────────────────────┘
```

## 风险控制

- **单笔风险**：每笔交易风险不超过初始资金的 1%
- **手数限制**：最小 0.01 手，最大 1.0 手
- **回撤限制**：当日回撤超过 20% 自动停止
- **自动止损**：每单固定 20 点止损

## 注意事项

⚠️ **重要提示**：

1. 此工具目前使用模拟价格，**不可用于真实交易**
2. 实盘使用前必须完成 MT4 桥接集成
3. 建议先在模拟账户充分测试
4. 量化交易有风险，入市需谨慎
